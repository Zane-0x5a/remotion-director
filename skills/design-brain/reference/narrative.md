# 叙事骨架(先立骨架,美学在它之上长)

> 叙事是美学的骨架。先产出整条脊柱:每拍**本拍发生的事**(观众看见**什么**发生——一个视觉事件 / 呈现机制;**文案**常是其中承重的一种载荷) + **运动张力(-2..2 的强弱意图)** + 该拍在整体里的**权重与角色**。

## 6 条判准(生成端**选择函数**,不是事后筛子)

编脊柱时**落笔就用它们当"什么算赢"**(质量杠杆 = 把"什么算赢"讲清楚,不是教不会的模型编脊柱——它会):

1. **交付实质性**:每拍把观众内部状态的 Δ **真做出来/导致**,不是仅陈述。("专注涨了 3 倍"要让观众**感到**那个跃迁,不是念出"3 倍"。)
2. **强度-权重配比**:视听强度匹配本拍在整体里的权重。**不足 = 怯懦;挥霍 / 偷走后续峰 = 灾难**(两向都罚)。
3. **信噪比**:剔掉无功能元素后,单一核心信号在**本拍时长 + 手机距离**内无阻力秒懂。
4. **跃迁可信度**:X→Y 的幅度在观众接受阈值内,不强行按头给没准备好的情绪。
5. **释放时机**:Δ 的交付点踩在注意力衰退临界**之前**,无拖沓、无过早泄洪。
6. **动量继承**:起点接住上一拍的势能(确认 X),非无意义断裂 / 重启。

## 叙事自审(只"验证",绝不"预设形状")

**机械禁令**:自审问句里出现下列任一词即非法、删掉——
`第N秒 / 起承转合 / 高潮(该在哪) / 铺垫 / 收尾 / 转折点 / 第几拍 / 曲线 / 弧线 / 该上升 / 该下降 / 最后两拍 / 开场必须 / 结尾必须 / Peak-End`。
理由:这些是"预设地形"(古典模板后门);判准只许验证"已生成的东西自不自洽",不许规定"它该长什么形"。合法形态只有三族:

- **相邻对比**:相邻两拍(运动张力, 空间构图, 节奏)是否完全撞车?撞了且非声明的 rupture → 必须变轨至少一轴(否则"换拍"在画面上无差分 = 一次替换不是一次过渡)。
- **极值自洽**:标了张力极值(或被声明为峰)的那拍,**本拍交付的东西**(文案、或被演出的视觉事件,或两者)扛不扛得起全片情绪/信息极值?扛不起 → 降级该拍(不规定峰该在哪,只验声明的峰兑没兑现)。
- **文案↔意图矛盾**:**当本拍有文案时**,文案的字面性质和本拍的构图/节奏意图打不打脸?(列举内容配 isolate、对立内容配 converge、单句配满版网格、CTA 配最低权重、疑问句配速断……)

## 方法论:编一根**有挣来 variance** 的脊柱

- 目标 = 哪拍重 / 峰在哪 / 停顿给谁 / 动量怎么继承——这是脊柱的**形状**,由叙事决定,不是从模板套。
- 门槛 = 中等带胜任(非杰作):脊柱自身形状胜任度(每拍声明的权重有没被内容挣到、整体平不平)。**绝不拿"匹配某根特定愿景弧"当尺。**
- **一次性生成整条脊柱**(一次性生成本就有跨拍上下文,不需波形模板硬编码 arc 数组)。
- 顶部"有没有灵魂"= 真品味墙,接受中等带上界,不假装到杰作。

## 交接美学

骨架定了再进美学。带过去的是:每拍**本拍发生的事**(被演出的视觉事件 / 呈现机制 + 其上的文案载荷)/ 运动张力 / 权重角色 + 全片节奏感(哪拍该静、哪拍该炸)。美学逐拍在这个骨架上长——叙事先想清楚,美学在它之上长,不是另起一摊。
