# 节奏 / 时序设计

> 管"拍内多元素的相对时序"——谁先谁后出场、各停多久、谁让谁。你亲手定每个元素的 enter/hold/exit 帧;下面是落笔时记住的机制。

## 三种并存策略(同时决定"屏上摆几个")

- **同位轮换(solo)**:同一位置,元素逐个替换,任意时刻屏上 1 个。摆 1 个。
- **排死逐亮(reserve)**:位置预排好、全程不动,元素逐个点亮、累加(列表型)。摆 N 个静态全景。
- **流动顶走(reflow)**:新元素进、旧元素平滑让位(聊天流型)。几阶段版面 + 位移。谨慎用(最难做好)。

## 落笔记住

- **给够阅读时间**:每个文本要停留到读得完(中文 ~每秒 4-7 字,英文按词)。**别让一拍塞进读不完的内容**——超了拆拍/缩内容。
- **别让元素迟到**:一拍里若元素太多、最后一个出场太晚,它会被拍尾切掉、没有独处时间。控制一拍的元素数 × 停留预算 ≤ 拍长。
- **非同组别同时入场(common-fate)**:同时入场的元素被知觉绑成一组。语义不同组却同时蹦出 = 错误分组 → 让它们错峰(~5-10 帧可分辨间隔)。
- **焦点抢夺(跨轴)**:剧烈入场的元素在 0.5-1.5s 内强制抢走视觉焦点(外源性注意力捕获)。**最高静态权重的元素须配最高入场能量,或让抢眼的那个错峰到注意力转移的间隙。**
- **同时"在动/要读"的元素 ≤ 1-2**:concurrent 拍别同时让多个剧烈入场 + 要阅读 = 过载。凝视语义阅读同时只能 1 个。
- **均质动效是 tell**:别给每拍/每块套同一个入场。每个出场贴合它揭示的内容。
- **错峰类型当词汇**:你设计出场序时是在选一种错峰模式——零(同时)/ 匀速 / **加速(渐聚势)** / **减速(渐落定)** / **语义(最重要的先出、留最长间隔)**。默认别用匀速;按本拍叙事选。

## 退场 / 留存

每个元素默认待到拍尾(accumulate);提前退场仅两个正当理由——时间塞不下,或清场给重点(decay)。文字在被阅读的 hold 段必须完全静止(1px 漂移都算违规),运动只发生在入场/退场。

## 乐谱韵律(品味墙)

whisper→impact→whisper→impact→thesis 那种张弛快慢 = 叙事品味,不是可量化规则。由你逐拍的节奏基调承载。

## 跨拍过渡

拍间衔接靠:① 背景场的跨拍演化(冷→暖之类) ② 每元素的留存(留到拍尾 / 读完退)。所有拍连续地想、保证 beat N 退场接 beat N+1 入场的设计意图连贯。
