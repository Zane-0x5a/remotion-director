# 质感设计

> 你亲手写每个 CSS/SVG filter / canvas 数值——这是加材质那刻的判断工具。质感 = 让画面里**所有线索说同一种空间语言**(光从哪来、表面怎么响应、显示有什么缺陷),非滤镜后处理。生产 = headless Chromium/Skia。**下面所有数字都是可被 1080×1920 渲染帧像素证据推翻的初稿,别直接采信任何一个;无 P 级实证标注的一律当推断。未标注实现档 = T1(纯 CSS/SVG/Canvas 直写),WebGL-only 逐处已标。**

> **工序位置(自证,先读这段再读技法)**:本文档属于**美学/质感工序**——在叙事与 conceit 定稿**之后**才读入、才生效。质感服务已定稿的叙事:光与材质是场景中形体的**属性**,不是叙事**主角**。"非滤镜后处理"说的是质感必须与内容协同设计、不能末尾糊一层滤镜,**不是**"先定光、再定叙事"。反例(自证):8 路施工里 5 路把 conceit 第 0 步改写成"一池暖光当主角"(其中 4 路进一步把"文字住在光里"写进 conceit)——质感分上去了,设计/叙事分塌方;**"一池光"不是 conceit,是实现手段**。读完本文,只动执行层(光/材质/氛围的做法);conceit 与叙事主角一律不动——若感到想动它们,说明把工序读反了,停手。

## 光

光是质感总开关:**无方向的光 = 扁平**(3/3;[Q] 在负向清单显式列条)——**质感工序内部**先定光源逻辑,阴影/高光/衰减全服从它;而光源逻辑本身服从已定稿的 conceit("这束光照亮叙事里的什么"),不反过来。

- **接触阴影(3/3)**:暗色副本 + 极小模糊 + `multiply`,**只在贴边可见、核心区窄而高对比**,阴影色带环境色相**非纯黑**;伪 AO = 角落压扁径向渐变 multiply;第二级腔体/字腔遮蔽 = 半径略大、透明度更低([G])。初稿:模糊 1-5px,opacity 40-70%(伪 AO 15-50%),偏移 1-2px。判据:drop-shadow 远距纯灰=没接触感。
- **光要有色温梯度(2/3 [M][G])**:暖光是**色相渐变**(核心亮黄白→橙→红→边缘冷)非单一橙色平铺;被照亮层用**加法混合**(`screen` 最安全 / `add` 易过曝 / `color-dodge` 慎用),**绝不 multiply 叠光源**(光到之处反变暗)。初稿:光照层 opacity 15-60%。
- **衰减拟平方反比(单源 [M],作方向用)**:`radial-gradient` 用 **4-6 色标**手动拟 1/r²(核心集中、向外非线性快衰)。初稿:核心区 0-10% 处 opacity 0.8-1.0,衰减带 10-40%,尾部 40-100% 趋 0。色标太少退化成线性"手电筒感";衰减过快读作 vignette。WebGL 可 `1.0/(d*d)`(**WebGL-only**)。
- **伪 3D 光照(2/3 [Q][G])**:`feDiffuseLighting`+Alpha 高度图造漫反射明暗(Alpha 当高度合法,与下节 `feDisplacementMap` 读颜色通道是两套机制),`feSpecularLighting` 叠高光;**高光宽度 = roughness 代理**(金属窄亮高对比 / 织物宽软低对比)。初稿:azimuth 0-360°、elevation 0-90°、specularExponent 8-16(金属起点;[Q] 给标 I 非 P——推断非实测),surfaceScale 未给范围。判据:同种白细对称高光铺所有材质=塑料。

## 材质与噪声

- **Film grain(3/3)**:`feTurbulence`+`feColorMatrix` 转单色 + 低 opacity 叠加。**双功能** = 介质感 + 量化掩蔽(deband);深色场用**单色**(彩色噪声=业余刺眼);`numOctaves=1`(2/3)。进阶 **luminance-aware**(暗部安静/中间调活,单源 [G] 引 Unity)比全画面等强高级。**注:baseFrequency 三源域不重叠,整域是初稿,必须 1080×1920 实测标定,任何数字别直接信**:[M] 0.6-1.5(1.5 上界无据,业界 ≈0.6-1.0)/[Q] 0.001-0.05 /[G] "接近像素级到两三像素"。强度初稿:风格 grain 3-15%,纯 deband 时近不可见。判据:过高=沙子,过粗=压缩块。(**亮场注记**:亮底对脏点极敏感,低频/彩色/过大颗粒=灰尘污渍,纸纤要方向性——见高调章纸纤/去banding 条。)
- **纸感/印刷感(3/3)**:三类弱信号**同时**出现才成立——低对比纤维 + dot gain(墨迹浸润 / `feMorphology` dilate)+ 网点痕迹;**内容本身必须响应不规则性,只做背景纹理会脱节**。实现:纸面层 feTurbulence(multiply)+ 油墨层 feDisplacementMap 扭曲边缘。**注:feDisplacementMap 读颜色通道值(R/G/B/A 经 `x/yChannelSelector` 选,0.5=零位移),非"高度信息"**(高度图归 feDiffuse/feSpecular),须显式设 selector。初稿:dilate radius 0.1-0.8,displacement scale 1-5(高度敏感),纸纹 opacity 5-10%。判据:纸纹对比过高→先看到纸;dot gain 做成外发光→特效字。(**亮场反转**:亮底纸感走方向性微纹**非**浅灰纹理叠加[抬背景亮度削对比=灰闷],见高调章纸纤条。)
- **材质暗示(2/3,金属/玻璃/织物)**:金属 = 窄亮高对比高光 + 拉丝 feTurbulence(`baseFrequency='0.01 0.5'` 不对称拉伸);玻璃 = `backdrop-filter` 模糊 + **边缘全反射高光线**(无则只是 frosted modal),或大 blur 负 spread `box-shadow`([Q],不依赖 backdrop-filter);织物 = 细密噪声(bf>1)+ 大面积 `soft-light`。**暗示必须微妙,过度即拟物按钮风**;可行域三源未给全,初稿实测。**注(项目级外部约束,非三源)**:玻璃当**默认装饰风格** = absolute ban(aesthetic.md),仅叙事需求下罕用。

## 大气与深度

- **大气透视(3/3)**:远层 blur + 降 contrast + 降 saturate(+略偏冷);**2D 里用前/中/后景各自的对比/饱和/blur 量代理深度**,可插半透明环境光色雾层。初稿:blur 0.5-4px,contrast 100→70%,saturate 100→50%。判据:层次不递进=读不出深度;全画面统一降饱和加灰="糊"非"远"。
- **Bloom / 光晕 / Halation(3/3)**:成立的 bloom 靠**显式阈值掩模**(只最亮部分参与)+ 多尺度 blur + `screen` 叠回;多层(肥厚亮边/泛光/极低 opacity 大半径)比单层可信;**halation** = 高对比亮边上的暖色窄 halo([G])。普通 UI 白字**绝对禁用**。初稿(三源收敛概念,参数来源未细分全视初稿):模糊半径=物体 50-300%,brightness 120-200%,叠加层 opacity 50-100%;**阈值"最亮 5-15%"无来源(未核查 Epic),只当方向性**。判据:全员辉光=无焦点;发灰=能量不足。(**亮场注记**:广域 bloom 是亮场最廉价"做柔",先吃边缘清晰度再吃材质分离→亮场大幅收缩,见高调章。)
- **Vignette(2/3 [Q][G])**:全屏径向渐变中心透明→边缘暗;**当注意力收束工具**(发力在边缘中段)优于当情绪滤镜([G] 区分两用法,引 Epic)。初稿:边缘 opacity 约 0.1([Q] 单源凭空数,深色场从低端起、不自造上界、实测)。判据:过度暗角压垮暗部;过强=模板滤镜感。
- **景深 DOF**:**注:前景/背景模糊比无定值常数**("1.5-2 倍"是编造,CoC 连续函数且方向可反)——别套固定比例,按"哪层是焦点"定模糊量;可行替代 = 大气透视 + 层级 blur 梯度([G] 间接,T1)。

## 暗端与渐变纪律(深色场命脉)

- **暗端管理(3/3)**:用**带色相深色**(`hsl(240,10%,5%)`/`#0a0a0f`)替代纯黑;大底**抬离绝对黑**预留空间给 grain/vignette/haze/接触阴影,**绝对黑只钉轮廓/孔洞/缝隙锚点**(ACES toe 的 2D 类比);提亮暗部须**同时注入冷色相(蓝)**非单纯提灰。初稿:最暗点 L(HSL) 3-10% / RGB<15,调整层 opacity 10-30%。判据:**死黑上强推 bloom/grain → 无过渡空间,后期痕迹异常显眼**。(**亮场对偶**:白端的"白点预留"见高调章,管最亮点不死白。)
- **Banding 三件套(3/3)**:① 叠极低强度低频 feTurbulence(opacity 1-5%,screen/overlay)② 增色标缩短插值距离 ③ Oklab/Lab 插值;**深色渐变在 OLED 上极度危险**。**注:banding 无绝对百分比阈值,依赖观察条件/gamma/色相**(三源一致;[M]"低于 10% 易 banding"核查无据,弃)。判据:抖动过强/过粗=新伪影;Bayer/规则网点 deband→规则纹。
- **Mach 带(2/3 [Q][G])**:banding(量化)与 Mach bands(知觉——渐变拐点处人眼看到额外亮/暗线)是**两个独立病因**;Mach 带须靠**削平坡度斜率突变**修,光叠 dither 不够。真实视知觉事实(Mach 1865 / Palmer / Wallis & Georgeson 2012)——**这几处文献出处未经独立核查,用到须自行核对原始文献,不得直接当定论引用**。
- **颜色空间分工(单源 [G],高价值)**:**光学运算**(发光/模糊/specular/遮蔽)走 `linearRGB`;**设计插值**(品牌渐变/氛围色带)走 Oklab/OKLCH。注意 **SVG filter 默认 linearRGB、SVG 渐变默认 sRGB**(`color-interpolation-filters`,MDN)。判据:全走同一空间 → 光学混合发灰发脏,或色带暴露 banding。

## 高调亮场(亮场命脉)

亮场的认知入口是一记反转:**暗场点亮事物靠绝对亮度,亮场区分事物靠相对对比**([G²][Q²] 2/2)。背景自带高亮度后,可见性全靠相对差(韦伯 ΔI/I≈K:背景越亮,可察觉所需增量越大)——**任何无意中降对比的操作在亮场都致命**(叠浅灰、压脏灰、广域 blur 都走这条死路)。这是暗端"抬黑/带色相深色"在白端的对偶:那边管最暗点不死黑,这边管最亮点不死白。

- **白点预留(亮场对偶,[G²] 单源 I+P,参数 S)**:工作白落 `OKLCH L≈0.90-0.98`;同画面相邻白层级 `ΔL≈0.01-0.05`;**逼近显示白(L→1.0 截断)只许极小面积镜面高光**。底色/渐变优先在 oklab/oklch 或 srgb-linear 插值(SVG filter 原语默认 linearRGB,对亮端有利)。判据:整屏 #FFF 死白 / 怕死白反压脏灰=双极失败。
- **接触阴影(亮场升级版,[G²][Q²] 2/2 I+P,参数 S)**:亮场长投影直接读成 UI 浮卡——**杀长投影,只留接触处极局部暗化**(峰值压暗约 1-6%,能量集中接触边不外飘);机制=在前景脚下造局部微型"暗场",以对比而非绝对亮度传达接触。这是上文接触阴影条的亮场分支(暗场可宽松用外投影,亮场默认大投影必塌)。判据:系统默认大灰影=主体漂浮。
- **边缘微差(白上白塑形,[G²] 单源 P+I,参数 S)**:近白对近白的塑形靠轮廓邻域窄带微差(内侧略暗/略冷,外侧背景略暗/略暖),`ΔL≈0.01-0.04`,色相偏移再小于亮度;Mach band/Cornsweet 把窄过渡放大成面差。判据:做成描边/四边等宽/无光向=失败。
- **边缘反向分离(白物对白底,[G²] 单源 I+P,参数 S)**:分离发生在轮廓附近(背景圈 multiply 略暗 **或** 主体边缘极弱 screen/specular,取一为主),不在主体下方一团大灰;单向光只做受影响的一两侧。这是 R1 暗场"深色描边/外发光"在亮场的替代正解(亮底深描边失效=孤立深斑+刺眼边界,[Q²] B 条目同向)。判据:误做成描边/每边同强度=失败。
- **镜面/漫反射分离([G²] 单源 I+P,参数 S)**:亮底出材质不靠整面变暗,靠 specular 组织:**高峰值、低覆盖率**("只有几条对的位置会亮"),或干脆纯漫反射;feSpecularLighting 窄高光 + arithmetic composite。判据:全域塑料光/宽奶白高光/边缘 halo=失败。
- **涂布/未涂布分流([G²] 单源 I+P[75° 纸面光泽=TAPPI T480 ✓],参数 S)**:做亮纸感先选分支:未涂布=diffuse 主导 + 极弱方向性纤维 + 软墨缘(dot gain 更明显);涂布=干净底 + 少纤维 + 偶发窄镜面。两头都沾必塌。
- **纸纤方向性微纹([G²] 单源 I+S)**:feTurbulence 只是零资产起步(本质 Perlin=云/大理石),必须叠方向性模糊/拉伸才像纸纤;**方向各向异性 > 整体对比度**;纹理低到"不放大几乎不意识到在",只在大块亮平面防死白。判据:云纹/霉斑/可数沙点=失败。(与上文 Film grain 联动:亮底对脏点极敏感,低频/彩色/过大颗粒=灰尘污渍。)
- **亲纸凹凸与字墨([G²] 单源 I+P[letterpress kiss ✓],参数 S)**:传统 letterpress 目标是"kiss"不留压痕,深咬印是现代复兴风→**默认浅压浅凹轻暗边,只有签名元素才明显加深**;feDiffuseLighting 用 alpha 做 bump;文字 multiply 深中性色 + 边缘轻软化。判据:深咬成皮革/金属 bevel/近白区堆假网点=失败。
- **白上白→微浮雕 + 微光泽差([G²] 单源 I+S)**:近白区色相差不是主线索,形体 + 反射组织才是:先 feDiffuseLighting 微地形,再决定是否 feSpecularLighting 局部 gloss 差,镜面永远辅佐凹凸。**白墨工艺数值 [G²] 主动拒报(无稳妥一手资料),真空勿脑补**(见"待补勿脑补")。
- **暖亮场三层白([G²] 单源 I,参数 S;暖偏色 2/2)**:基底白(纸感)/受光白(空气)/遮蔽白(贴附重量)三层都在才是纸面或晨光,缺层=偏色滤镜;白区色度"有倾向不做有颜色",暖偏移弱于亮度层级。**暖偏色=条件性成立**(R1 光色温梯度的亮场延伸,[G²][Q²] 2/2):直接 hue-rotate 塑料感,正解=低色度 + 渐变叠加/LUT。判据:满屏奶黄/旧纸滤镜/暖过头脏感=失败。
- **中性白锚点(冷调命门,[G²] 单源 I+P,参数 S)**:冷亮场不是"更蓝",是**先有不染色的白锚点**,再把远景/背光侧轻微推冷(同时略减对比与色度),主体接触处回中性;冷偏移永远弱于亮度对比。判据:全域蓝洗/医院灯管/死冷=失败。
- **银灰窄镜面(冷调,[G²] 单源 I+P,参数 S)**:冷亮场避"死平廉价"靠窄而准的镜面条 + 边缘能量(主体面大多中性平静,只在曲率合适处细长高光),不靠加灰;高光宽度显著窄于明度坡度,覆盖率低。判据:边缘一圈通亮/宽奶油高光=石膏或塑料。
- **临床白三配比(冷调,[G²] 单源 I+S)**:冷调成立靠三件配比=中性锚点 + 可读接触线索 + 极少量高光组织,三缺一即滑向"骨架屏白";背景总梯度比暖场更收,冷调只落远处与背光侧。
- **定向渐变纪律([G²] 单源 I+P,参数 S)**:近白背景全平=死,无据中心径向光=假;最稳=单一主方向大尺度渐变(像窗光),主梯度总 `ΔL≈0.01-0.06`,次级修正光更弱且方向不互打;stop 导数变化尽量平(Mach band 把拐点放大成假轮廓)。判据:中间亮四角暗的"预设壁纸感"=失败(替代上文 Vignette 中心径向用法:亮场无物理依据中心亮晕=模板背景)。
- **指数式大气透视([G²] 单源 P+I[指数衰减=Koschmieder/Beer-Lambert ✓])**:按深度/分层做指数式 transmittance,优先衰减中高频对比;前景接触关系最后才被吃掉。**"对比先降→饱和后降"次序句仅标 S**(文献均作同机制并行产物,无分阶段实证)。判据:线性白雾盖板压扁空间=失败。(上文大气透视条的亮场注记:亮底大面积 backdrop blur 吃掉全部纹理=灰蒙无生气[Q² S1 2/2],用指数衰减/有方向的径向渐变[**非**无据中心 vignette,见定向渐变条]/微噪点替代。)
- **亮端去 banding([G²][Q²] 2/2 P+I,参数 S)**:梯度在 oklab/srgb-linear 定义 + 输出前叠极细抖动;抖动走亮度通道、色度噪声压低、幅度刚够打散台阶;**Canvas 程序化 hash/noise 比 feTurbulence 更适合 deband**(Perlin 非蓝噪)。**近白 banding=亮场头号危险**(2/2)。判据:彩色砂点/云纹假纹理/deband 做成可见 grain=失败。
- **亮底深字=墨落材质([G²][Q²] 2/2 I+P[W3C Silver 研究 ✓非规范])**:字色入材质体系:暖纸偏暖黑/冷白偏冷黑 + 极弱 multiply 复制层或轻局部底洗;**禁白 glow**(R1 暗场反白字 halo/光晕补字法在亮场反转,白 glow=脏边);**禁纯黑洞眼**(深中性墨色替代;"WCAG 推荐 #212121"是 [Q²] 编造归因[refuted],#212121=Material Grey 900,该单点色值弃用,"亮底勿纯黑"主张本身 2/2 保留);字号越小字重越细对比越要上推。
- **亮场动画 grain/shimmer([G²] 单源 P+I[STBN ✓:NVIDIA Wolfe et al. EGSR 2022,时空两域蓝噪])**:STBN 最佳但 **WebGL-only**(依赖白名单外);白名单内退化=**噪声锚定屏幕或世界 + 慢速相位 + 绝不每帧重 seed**;近白渐变运动速度低到量化台阶不相邻帧跳格。判据:亮底发闪/细噪爬行/边缘 shimmer=失败。(上文运动章 grain 动画条的亮场注记:亮底时域白噪极显眼,比暗场更严。)
- **亮场工程基线(2/2 杂项,实现层)**:`ctx.filter` 非 Baseline(Safari 18 实现但默认关,headless Chromium 可用 ✓);SVG `color-interpolation-filters` 默认 linearRGB,要 sRGB 运算须显式声明([Q²] 点名,与上文颜色空间分工一致);blend mode 亮场风险大于 soft-light([Q²] caution,与上文三分法兼容)。**[Q²] 三条 refuted 已剔除不入装备(勘误在场)**:① `premultipliedAlpha=false`(WebGL 专属 attribute,Canvas 2D 设置=静默 no-op,无此开关);② "降 baseFrequency 取高频细节"(方向写反,bf 越低噪点越粗,要细=提 bf);③ WCAG#212121 编造归因(见上条深字)。

## 边缘与混合模式

- **混合模式语义三分法(单源 [M],全批最有价值判断工具)**:落 blend mode 前先归类——**光类**(screen/add/color-dodge/overlay/soft-light/hard-light = 加法物理,光源图层**必须**用)/ **影色类**(multiply/color-burn/darken = 减法阴影)/ **滤镜类**(difference/exclusion/hue/sat/color/luminosity = 风格化)。[G] 反证:全局 overlay/hard-light 重写整幅 tonal、不带空间信息 = 读作滤镜。判据:multiply 叠光源 = 光所到之处变暗 → 选错类。
- **边缘品质三分(3/3)**:硬边=金属/玻璃/图形(矢量/border);软边=织物/云雾/失焦(blur);不规则边=有机/手绘(feDisplacementMap+feTurbulence)。混用丰富层次,但**边缘品质须与材质暗示一致**(金属+软边=认知混乱)。

## 文字的质感

- **文字的环境响应(常住暗带;3/3)**:字**摆在哪里**由构图与叙事定——常住暗带、靠局部暗化/吸收坐进场景,**绝不默认搬进亮核**(反模式,自证,8 路里 6-7 路中招:把文字搬进光核/画面最亮区=对比塌陷,可读性直接死)。"坐进场景"只管字怎么**响应**环境:墨色响应(**从背景吸色**非纯白/黑)+ 轻微噪声(对比低于背景、静看才可见)+ 阴影与主光源一致;进阶**字 alpha 作 mask 裁切场景响应层**([G])。判据:纯白/黑文字在有色调场景=刺眼孤立;默认 `text-shadow(2px 2px 4px #888)`=廉价代名词。
- **opsz + 反白字 lint(单源 [G],可进 validator)**:`font-optical-sizing:auto`+variable font opsz 轴——小尺寸变厚更开,大尺寸允许高对比纤细;**暗底小字优先 opsz/字形,而非先上描边重阴影**。反白字 6 项 lint:opsz 开了吗 / 尺寸×细权重危险组合 / 高对比 serif / tracking 过紧 / 叠 blur-glow / 无局部底垫。

## 运动中的质感

- **grain 动画(3/3)**:静态 screen-space grain = 磨砂膜(好),纯逐帧白噪声 = 雪花(坏)。三法互补:[M] `frame%N` seed / 滚动 200% 大纹理每帧移 1-3px;[Q] **hue-rotate 0-360° 无缝循环** + grain 容器随主体移动(非固定背景);[G] 6-16 帧短循环蓝噪声 tile + 锁对象坐标(低频材质)/锁屏幕坐标(高频 deband)双层。判据:shimmer/crawl = 运动边缘蚂蚁线;高频闪烁 = 逐帧全随机。(WebGL STBN 最理想,**WebGL-only**;NVIDIA STBN 归因已核查 ✓=Wolfe et al. EGSR 2022 时空两域蓝噪,详高调章亮场动画条;**亮底时域白噪比暗场更显眼,绝不每帧重 seed**。)
- **运动模糊(3/3)**:**优先 temporal supersampling(子帧累计)**,而非单帧后期方向 blur(后者只近似匀速直线可用)。手工 = **复制 3-10 份沿运动反向 + 降透明度(0.8→0.2)+ 轻 blur**;只在足够快速度出现,**禁止全局 blur**,逐物体 + 速度相关。**注:绝不存在 remotion 核心的 `<MotionBlur/>` 组件或 `motionBlur` 属性**;正确 API = `@remotion/motion-blur` 包的 `<Trail>`/`<CameraMotionBlur>`,**但该包不在当前依赖白名单**,本环境手法 = 上面的多副本拖尾或子帧思路。判据:方向 blur 套在阴影/文字/glow 上 → 次级运动不同步。

## 负向清单:读作廉价的 14 种机制(match-and-refuse)

伸手要拿任一条 → 换服务 conceit 的真机制重做。[G] 反证式校验(**关掉 X 看损失,几乎没损失=原本过量**)是逮捕工具。**[自证]** = 自家盲评出生证实验像素实证(n=12 全批审计,质感臂 8 路为主要逮捕样本,人眼复核;非三源研究)。

| 廉价源 | 机制(为什么廉价) | 反证 test |
|---|---|---|
| **均匀辉光 / 无阈值 glow**(3/3) | 无亮度中心/无非线性衰减,所有亮度都散射 → 贴纸 | 关掉非最亮区 glow 几乎没损失=过量 |
| **默认投影 / drop shadow**(3/3) | 右下纯灰半透,无视光向/环境色/距离 → UI 库组件 | 换接触阴影更贴地=原 shadow 是装饰 |
| **塑料渐变**(3/3) | 高饱和大色相线性插值,缺协同+粗糙度,单层对称高光 → 失材质依据 | 遮高光后材质判断不变=高光没传材质 |
| **枕形浮雕 / 对称 bevel**(2/3) | 不该同亮的边同加亮暗,违反单一光照先验;与 90 末拟物 UI 绑定 | 只留一个方向明暗,矛盾消失=原本是假 |
| **处处同强度 grain**(3/3) | 不随亮度/材质响应 → 叠层滤镜非介质 | 对比 luminance-aware 版,等强版立显机械 |
| **#000 死填**(3/3) | 真问题是技术坑不是"暗":8-bit 近黑高发 banding + 无空间容纳叠加物——暗场本身没错(有重心/服务高对比的深场是对的),错在踩了 `#000` 这个值的两个技术坑 | 抬到带色相深色后 banding 消失 + 暗端现层次=原本踩的是 #000 技术坑,非"太暗" |
| **无方向的光**(单源 [Q]) | 无方位角/仰角 → 物面无明暗分区,扁平 | 问"光从哪来"答不出=没光逻辑(test 为推断,原文只给机制) |
| **无接触阴影**(单源 [Q]) | 缺遮蔽暗区 → 漂浮空中,缺物理根基 | 遮底边交界分不清谁在前=缺接触阴影(test 为推断,原文只给机制) |
| **全局 overlay/hard-light 调色**(单源 [G]) | 重写整幅 tonal 不带空间位置 → 全局滤镜 | 关掉只"没那么有调"无局部光丢失=滤镜 |
| **细而规则 halftone / 线性纹**(单源 [G]) | 与像素栅格/采样相撞 moiré/crawl,尤甚手机+运动 | 运动几帧看边缘是否爬行/出摩尔纹 |
| **反白细衬线小字**(单源 [G]) | 暗底渗散/磨损,counters 可见度降,视觉+工艺双脆弱 | 跑反白字 6 项 lint,命中即换 opsz/字重/底垫 |
| **大气光球 / 无锚光污染**([自证]) | 光只与"空气"互动:大面积辉光不落在任何形体上——没有被照亮的表面、没有遮挡、没有进入场景几何的衰减。**有亮核-衰减剖面也照样中招**:剖面对了但什么都没照亮 = 氛围填充非质感(与"均匀辉光"条互补:那条逮无结构,这条逮有结构但无对象) | 问"这束光照亮了画面里的哪个形体?"答不出=光污染;关掉它,叙事毫发无损=原本只是雾 |
| **文字压光核**([自证]) | 文字摆进辉光最亮区,衬底亮度逼近字色 → 对比塌陷;"字坐进光里"的字面误读(正解见上文"文字的环境响应":响应环境,常住暗带) | 量字与衬底的亮度差;字底落在全帧最亮区且字色同向=中招 |
| **大面积单色死填(任意色,非黑也算)**([自证]) | 大块均匀色区无光照响应、无材质结构——数字填充非物质表面(逮的是"均匀填充=非物质",不论色相;与 `#000` 条共享这一半,#000 另背近黑 banding/无 headroom 两条专有技术坑)。注:有重心/服务高对比的留白不在此列,这条逮的是"该有材质结构的表面被填成了平板",不是"色块面积大" | 区内多点取样,RGB 几乎无变化**且无 grain/渐变结构**=死填;补上光照梯度后空间感立现=原本是填充 |

## 组合思路 + 配方

质感不是叠 20 个 filter,是让**微观颗粒 / 中观材质暗示 / 宏观环境空间**协同([M] 三层)——"噪声给材质附着面,光照决定材质被看见"。落笔用 [G] **三问**自检:① 光学现象(光从哪来、怎么衰减)② 材质表面(边缘品质与材质暗示一致吗)③ 显示缺陷(banding/grain/暗端处理了吗),**静帧和运动各审一遍**。

配方=已有技法重组(示例非模板,技法全在上文,混合模式按三分法选类,「塌点」= 先暴露的一环):暖窗(dark,塌点=同亮同色同强 bloom→贴纸灯箱)、金属铭牌(dark,塌点=单层/同频高光→塑料奖牌)、油墨公告(中调,塌点=纸纹过强先看到纸 / 渗边做外发光)、玻璃牌匾(中调,塌点=只 backdrop blur 无边缘高光→iOS modal)、CRT 余晖(dark)。

高调亮场三配方(high,[G²] 三张,叠层次序见高调章各条;"示例非模板"原文自带):暖纸编辑亮场(high,序=工作白底→单向暖洗光→方向性纤维→极浅盲压→墨字→接触暗化→去banding 细抖动;塌点=纤维+暖偏色过量→"旧纸滤镜")、冷白产品亮场(high,序=中性白锚点→极弱冷向梯度→轮廓反向分离带→窄镜面条→极局部接触阴影→去banding;塌点=为"显出白物"压灰背景或描边→廉价)、雾天自然高调(high,序=高位天空白→指数式对比衰减→远处轻冷偏→前景保接触→静态/慢相位抖动;塌点=雾做成线性白盖板→深度全平)。

**待补勿脑补**:① 高调亮场配方——**已由 R2 双源补课**(见上文"高调亮场(亮场命脉)"章);残余真空续挂(吸收时仍不得脑补):白上白数码白墨工艺数值([G²] 主动拒报,屏幕近似=微浮雕+微光泽差已给)/ grain 可见性随明度变化的定量刻度(双源只给方向,无 L→参数域定量函数,照旧像素实测标定)/ 真 STBN 在依赖白名单内不可用(WebGL-only,退化策略已给)/ [Q²] 缺席的 ②③④⑤⑧ 五维全单源([G²] 主张级抽核过但无跨源验证,徽章如实标单源)/ 高调亮场自家像素病例锚稀缺(病例库以暗场/中调为主,逮捕率验证若做正锚薄须声明)。② demoscene/shader art 可移植算法、AE/motion 社区图层堆叠惯例——需要时起独立研究。③ 预编译/外部资产、WebGL-only 技法——本档暂无系统覆盖,需要时起独立研究补录;**如有能力、有需要可自行尝试实现。**

> 真品味墙(grain 落 3% 还是 8% / 暖窗 vs 冷调 conceit / 拉丝方向 / 暗角强度 / 纸纹粗细)——可行域内残余,无客观最优,按整体风格定调。**唯一不可推翻的是机制层**(光要有方向、grain 要 luminance-aware、暗端别死黑、混合模式选对类);所有数字都是初稿,以 1080×1920 渲染帧像素证据为准。
